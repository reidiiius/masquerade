### Usage
Command-line interface

    cd masquerade/

:

    julia bandana.jl

:

    julia bandana.jl n0

:

    julia bandana.jl j6 k5

:

    julia bandana.jl cello j6 k5

:

    julia bandana.jl bass every

REPL

    julia

:

    include("bandana.jl")

:

    Bandana.entryway()

:

    Bandana.entryway("n0")

:

    Bandana.entryway("cello", "j6", "k5")

:

    Bandana.entryway("bass", "every")

:

    exit()
